CSS Tutorial

This tutorial provides a quick reference to commonly used CSS properties and their values.

Selectors

  • * - Universal selector
  • element - Selects all elements of that type (e.g., p, div)
  • .class - Selects all elements with the specified class
  • #id - Selects the element with the specified ID
  • [attribute] - Selects elements with the specified attribute

Box Model Properties

  • width, height - Set the width and height of an element
  • padding - Space between content and border
  • border - Border around the element (e.g., 1px solid black)
  • margin - Space outside the border of an element

Typography Properties

  • font-family - Specifies the font of an element (e.g., 'Arial', sans-serif')
  • font-size - Sets the size of the font (e.g., 16px)
  • color - Sets the color of text (e.g., #333333)
  • line-height - Sets the height of a line of text (e.g., 1.5)

Background Properties

  • background-color - Sets the background color of an element (e.g., #f0f0f0)
  • background-image - Sets a background image (e.g., url('image.jpg'))
  • background-size - Specifies the size of the background image (e.g., cover)
  • background-position - Sets the initial position of the background image (e.g., center center)

Flexbox Properties

  • display: flex - Defines a flex container
  • flex-direction - Specifies the direction of the flex items (e.g., row, column)
  • justify-content - Aligns flex items along the main axis (e.g., center, space-between)
  • align-items - Aligns flex items along the cross axis (e.g., center, stretch)

Grid Properties

  • display: grid - Defines a grid container
  • grid-template-columns - Specifies the number of columns and their widths (e.g., 1fr 2fr)
  • grid-template-rows - Specifies the number of rows and their heights (e.g., 100px 200px)
  • gap - Sets the gap between grid items (e.g., 10px)

Positioning Properties

  • position - Specifies the type of positioning (e.g., static, relative, absolute, fixed)
  • top, right, bottom, left - Specifies the offset of an element from its normal position (used with positioned elements)
  • z-index - Controls the stacking order of elements (higher values are in front)
-- Css usage examples --

Example: Centering a Div

.centered {
    display: flex; 
    justify-content: center;
    align-items: center;
    height: 200px;
}

This CSS class will center any content inside a div with the class "centered".

Example: Creating a Responsive Grid

.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}

This CSS class will create a responsive grid layout with columns that adjust based on the available space.
